# **CRUD com Node.js**
Temos aqui uma passagem pelo processo de um CRUD e um login com segurança!

Utilização não apenas do node.js para o backend, mas ferramentas essenciais 
como por exemplo babel, para que outros navegadores também possam rodar a aplicação sem problemas de compatibilidade,
e EJS engine de templates para Node.js que permite gerar HTML dinâmico incorporando código JavaScript diretamente em arquivos de template.

Contém a arquitetura MVC dividindo modulos e suas funcionalidades!

**Para executar a aplicação inicie o node e instale os pacotes:
```bash
# npm init
```
depois execute:
```bash
# npm install
```
A url para abrir o servidor local é localhost:3000
